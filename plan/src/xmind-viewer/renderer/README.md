# 源码渲染器

## 能力定位

`src/xmind-viewer/renderer/` 保存源码版 XMind viewer 的布局和 SVG 渲染逻辑。它替代旧 Snowbrush bundle 的主渲染入口，但当前目标是可维护的只读脑图查看，不追求完整复刻 XMind 桌面端或 Web 端。

## 当前模块

- `layout.ts`：把 topic 树转换成左右分布的节点坐标、尺寸、文本换行和整体边界。
- `svg-renderer.ts`：根据布局生成 SVG 节点、连接线和文本，并提供 `setTransform()` 供缩放、平移和适配画布使用。

## 当前渲染口径

- 根主题居中显示。
- 一级主题左右分栏，子主题沿父方向展开。
- 节点宽高按文本估算，中文和英文使用不同字符宽度近似值。
- 连线使用三次贝塞尔曲线。
- 缩放、触摸板平移和适配画布通过 SVG group transform 完成。

## 限制

- 暂不支持完整主题样式、手绘结构、鱼骨图、矩阵图、时间线、组织结构图、关系线、边界、概要、图标、贴纸、图片、附件或公式。
- 后续补齐能力时，应先扩展文档模型，再扩展布局和渲染，不直接复制旧编译后 bundle。
