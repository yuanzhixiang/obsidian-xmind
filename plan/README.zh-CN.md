# 中文 README

## 功能定位

`README.zh-CN.md` 是中文说明文档，内容与英文 README 对齐，面向中文 Obsidian 用户和开发者，保持 `XMind Maps` 的极简项目介绍。

## 内容口径

- 中文为主，顶部提供 `README.md` 英文链接。
- 功能说明使用简洁单句：在 Obsidian vault 中直接打开 `.xmind` 文件，以只读模式查看脑图，支持缩放、适配画布和查看控件。
- 不维护安装、限制、使用、开发和发布章节。
- 保留 Apache-2.0 许可证与 `yuanzhixiang` 版权声明。

## 维护规则

- 中文 README 必须和英文 README 保持项目介绍、功能和许可证口径一致。
- 新增用户可见能力时，必须同步更新英文和中文 README。
