# Git Ignore 配置

## 功能定位

`.gitignore` 只忽略本地开发、调试、构建和编辑器产生的临时文件，避免把可再生成产物提交进仓库。

## 忽略口径

- `dist/`、`release/` 和 `bundle*` 是构建、打包或分析产物，不提交源码仓库。
- `.playwright-mcp` 是浏览器回归测试生成的临时目录，不提交源码仓库。
- `node_modules/`、测试 vault、Obsidian 热重载文件和本地编辑器配置不提交源码仓库。

## 约束

- 运行 `pnpm build` 或 `pnpm package` 后生成的产物用于本地验证，验证完成后可以删除。
- 新增忽略项必须确认不会屏蔽源码、plan 文档或发布所需的 Obsidian 标准三件套源文件。
